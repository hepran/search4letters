Requirements to reproduce this software are:
1. Python 3.
2. Flask.
3. These codes
4. A web browser

To run* this software (instructions), you need to:
1. Enter this command: python vsearch4web.py
2. In the terminal, there will a Uniform Resource Locator (URL) like "http://127.0.0.1:5000/".
3. Type in and search the URL in your web browse.
4. On the welcome page, enter your phrase and letters to search.
5. Click on the available button.

* I run this in Linux OS.
